# BossRush — gioca ora

**▶️ https://lorenzobacci20.github.io/bossrush-play/**

Boss rush 2D, giocabile da desktop (WASD + mouse, Shift/Spazio per il dash) e da telefono (joystick virtuale, auto-mira, ruota in orizzontale).

Questo repo contiene solo la build WebGL generata dal progetto Unity (repo privato).
